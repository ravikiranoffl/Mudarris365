## Lesson

1.  **Present Tense Recap: Prefixes and Root Letters**
    *   Remember that the Present Tense describes actions happening now or regularly.
    *   Unlike the Past Tense which mostly uses suffixes, the Present Tense primarily uses **prefixes** (letters added to the beginning of the verb) to show "who" is doing the action. The root letters will still be in the middle!
    *   Yesterday, we saw `يَـ` for "he" (e.g., `يَكْتُبُ`) and `تَـ` for "she" (e.g., `تَكْتُبُ`).

2.  **The Present Tense Verb for "I" (أَنَا)**
    *   To say **"I write"** or **"I am writing"**, we use the prefix **أَـ** (a-).
    *   **Structure:** `أَفْعَلُ` (af'alu) - This is a common pattern. It has a Fatha ( َ ) on the prefix, a Sukoon ( ْ ) on the first root letter, a Damma ( ُ ) on the second root letter, and a Damma ( ُ ) on the third root letter.
    *   **Example:** Let's take our root `ك ت ب` (meaning "writing").
        *   Add `أَـ` to the beginning: `أَكْتُبُ` (aktubu) - **I write / I am writing.**
            *   *Notice*: `أَـ` (prefix), `كْ` (first root letter with Sukoon), `تُ` (second root letter with Damma), `بُ` (third root letter with Damma).
    *   **Another example:** With the root `د ر س` (meaning "study").
        *   `أَدْرُسُ` (adrusu) - **I study / I am studying.**

3.  **The Present Tense Verb for "We" (نَحْنُ)**
    *   To say **"We write"** or **"We are writing"**, we use the prefix **نَـ** (na-).
    *   **Structure:** `نَفْعَلُ` (naf'alu) - The pattern is similar to the "I" form, just with a different prefix.
    *   **Example:**
        *   Take our root `ك ت ب`.
        *   Add `نَـ` to the beginning: `نَكْتُبُ` (naktubu) - **We write / We are writing.**
            *   *Notice*: `نَـ` (prefix), `كْ` (first root letter with Sukoon), `تُ` (second root letter with Damma), `بُ` (third root letter with Damma).
    *   **Another example:** With the root `د ر س`.
        *   `نَدْرُسُ` (nadrusu) - **We study / We are studying.**

4.  **The Present Tense Verb for "You" (Masculine Singular - أَنْتَ)**
    *   To say **"You (a man) write"** or **"You (a man) are writing"**, we use the prefix **تَـ** (ta-).
    *   **Important Note!**: This is the **exact same form** as the "She" (هِيَ) present tense verb you learned yesterday (`تَكْتُبُ`, `تَدْرُسُ`).
        *   **How to tell the difference?** Usually, the context of the sentence (who you are talking to, or if a specific feminine noun/pronoun like `هِيَ` is present) makes it clear whether it means "she" or "you (masculine)".
    *   **Example:**
        *   Take our root `ك ت ب`.
        *   Add `تَـ` to the beginning: `تَكْتُبُ` (taktubu) - **You (m) write / You (m) are writing.**
            *   *Notice*: `تَـ` (prefix), `كْ` (first root letter with Sukoon), `تُ` (second root letter with Damma), `بُ` (third root letter with Damma).
    *   **Another example:** With the root `د ر س`.
        *   `تَدْرُسُ` (tadrusu) - **You (m) study / You (m) are studying.**

5.  **The Present Tense Verb for "You" (Feminine Singular - أَنْتِ)**
    *   To say **"You (a woman) write"** or **"You (a woman) are writing"**, we use the prefix **تَـ** (ta-) AND add a special suffix.
    *   **Structure:** `تَـفْعَلِينَ` (taf'alīna) - The prefix `تَـ` (ta-), then the root, followed by a long `ـِي` (Yā) and `نَ` (na).
    *   **Example:**
        *   Take our root `ك ت ب`.
        *   Add `تَـ` to the beginning and `ينَ` to the end: `تَكْتُبِينَ` (taktubīna) - **You (f) write / You (f) are writing.**
            *   *Notice*: `تَـ` (prefix), `كْ` (first root letter with Sukoon), `تُ` (second root letter with Damma), `بِي` (third root letter with long Yā), `نَ` (Nun with Fatha).
    *   **Another example:** With the root `د ر س`.
        *   `تَدْرُسِينَ` (tadrusīna) - **You (f) study / You (f) are studying.**

This completes the most common singular and plural forms for Present Tense Form I verbs! You can now talk about who is doing an action in almost any basic context. These patterns are very consistent across thousands of Arabic verbs.

## Examples

Let's put these new present tense verb forms into practice with the pronouns you learned on Day 7, and some nouns you already know!

1.  **I write the book.**
    *   `أَنَا أَكْتُبُ الْكِتَابَ.` (Ana aktubu al-kitaaba.)
        *   *Notice*: `أَنَا` (I) matches `أَكْتُبُ` (I write). `الْكِتَابَ` (the book) is the object of the verb and takes a Fatha.

2.  **We study in the school.**
    *   `نَحْنُ نَدْرُسُ فِي الْمَدْرَسَةِ.` (Nahnu nadrusu fī al-madrasati.)
        *   *Notice*: `نَحْنُ` (We) matches `نَدْرُسُ` (We study). `فِي الْمَدْرَسَةِ` (in the school) uses our preposition rule.

3.  **You (masculine) write your (masculine) name.**
    *   `أَنْتَ تَكْتُبُ اِسْمَكَ.` (Anta taktubu ismaka.)
        *   *Notice*: `أَنْتَ` (You-m) matches `تَكْتُبُ` (You-m write). `اِسْمَكَ` (your name) uses an attached pronoun and takes a Fatha as the object.

4.  **You (feminine) study the new book.**
    *   `أَنْتِ تَدْرُسِينَ الْكِتَابَ الْجَدِيدَ.` (Anti tadrusīna al-kitaaba al-jadeeda.)
        *   *Notice*: `أَنْتِ` (You-f) matches `تَدْرُسِينَ` (You-f study). `الْكِتَابَ الْجَدِيدَ` (the new book) is a definite noun with a definite adjective, as learned on Day 11, and both take Fatha as the object.

5.  **I study my small car.** (Meaning I study about my car or check on it)
    *   `أَنَا أَدْرُسُ سَيَّارَتِي الصَّغِيرَةَ.` (Ana adrusu sayyaaratī as-sagheerata.)
        *   *Notice*: `سَيَّارَتِي الصَّغِيرَةَ` is a definite noun with an attached pronoun and a matching definite adjective. Both take Fatha as the object of the verb.

You have now learned the most essential present tense conjugations! This is a massive step towards holding actual conversations in Arabic.
